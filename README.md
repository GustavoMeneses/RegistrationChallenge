# RegistrationChallenge

### Pré-requisitos

Banco Postgres

* porta: 5432
* usuário: postgres
* senha: [sem senha]

Java 8

### Instalação

Clonar o repositório e seguir os passos:

Rodar os scripts de banco que estão no arquivo:

* database-scripts.sql

Rodar o pom.xml

```sh
$ cd backend/
$ mvn compile
$ mvn install
```

Instalar os pacotes via npm ou yarn

```sh
$ cd frontend/
$ npm install
$ npm start
```
